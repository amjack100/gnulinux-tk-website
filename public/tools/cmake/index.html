cmake

Tue, Jul 20, 2021

CMake is a tool that creates build setups for source code, allowing that code to be cross platform.

The root of a CMake entity is always defined by a CMakeLists.txt file.

  • Bare Minimum Example
  • File Templating

Bare Minimum Example

.
├── CMakeLists.txt
└── main.cpp

CMakeLists.txt

cmake_minimum_required(VERSION 3.10)

project(Main)

add_executable(Main main.cpp)

mkdir build
cd build && cmake ..

File Templating

CMake can format input files with variables specified in CMakeLists.txt, and produce an output in the build folder.

.
├── CMakeLists.txt
└── format-in.txt

CMakeLists.txt

configure_file(format-in.txt format-out.txt)

format-in.txt

The project version is: @Main_VERSION_MAJOR@
