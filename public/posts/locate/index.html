locate

Mon, Jul 19, 2021

locate is a tool for finding files extremely quickly. It uses a pre-generated database to get file locations instead of manually scanning the system with every use.

locate
run directly yes
configuration file N/A
service N/A

Most Common Usages

Find a file using a search pattern.

locate file

will scan the database for every entry containing the word “file”.

Show the number of matching files.

locate -c file
