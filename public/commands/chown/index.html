chown is a tool for changing the owner of a file or directory. To see the owner of file.txt, use the command ls -l file.txt. The output is:

-rw-rw-r-- 1 john staff 23 Jul 2 08:45 file.txt

where “john” is the file’s owner. chown is also capable of changing the file group, (in this case “staff”), doing the exact same thing as chgrp.

Most Common Usages

Change the owner of a file.

chown root ./file.txt

Change the owner and group of a file.

chown root:staff ./file.txt

Change the owner (and subfiles) of a directory

chown -hR root directory
