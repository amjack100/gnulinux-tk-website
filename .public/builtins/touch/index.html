touch

Thu, Jul 15, 2021

touch is command used to update the access and modification times of a file. It is most commonly used as a quick way of creating a new, empty file.

Usage Examples

Create an empty file.

$ touch file.txt

Update the timestamp of a file without creating a new one if it doesn’t exist.

$ touch -c existingfile.txt
