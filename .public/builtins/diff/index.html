diff

Sat, Aug 28, 2021

diff is a command for comparing files line by line. diff provides multiple output formats, making it useful as a command line tool as well as a scripting tool. The most common use case involves displaying changes of the same file across multiple versions.

Usage Examples

Compare two files.

~/one.txt

one
two
three

~/two.txt

one
two
four

$ diff ~/one.txt ~/two.txt
