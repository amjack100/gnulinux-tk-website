tty

Sun, Sep 26, 2021

tty is a minimal command that prints the file name of the terminal (emulator) which is connected to standard input. tty stands for teletypewriter.

Notes

  • TTYs (teletypewriters) and video terminals only exist in museums now, but jargon and abbreviations associated with them are still numerous in Linux commands and documentation.
