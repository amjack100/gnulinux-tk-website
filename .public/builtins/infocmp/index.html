infocmp

Wed, Sep 22, 2021

infocmp is a command used for printing out descriptions from the terminfo database, a database which contains terminal descriptions and data. The cmp in infocmp refers to the tool’s ability to compare data values across different terminals.

Note: for brevity, “terminal” and “terminal emulator” are interchangeable.

Usage Examples

Get terminfo data values corresponding to the current terminal.

$ infocmp

Get shared (common) data values between two terminals.

$ infocmp -c alacritty xterm

Get differing data values across two terminals.

$ infocmp -d alacritty xterm

Notes

  • The actual on-disk location of the terminfo database is provided in the first line of output, which will be similar to:

    #     Reconstructed via infocmp from file: /usr/share/terminfo/a/alacritty

  • See more about the terminfo database.
