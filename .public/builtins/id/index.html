id | whoami

Fri, Aug 27, 2021

id is a command which will display various information specific to a user, including UID (user ID) and GID (group ID). These are unique numbers that are used to determine which resources each user/group may or may not be given access to. whoami is a shortcut command that is identical to id -un.

Usage Examples

Display the user’s ID, group ID, and groups to which that user belongs.

Diplay the current user ID number.

$ id -u

Diplay the current group ID number.

$ id -g

Diplay the name of the current user.

$ whoami

Notes

  • The operating system keeps the User ID 0 and Group ID 0 reserved for the root user.

  • The same user information can also be found within the /etc/passwd file. You can see this with grep user /etc/passwd, replacing “user” with the actual user’s name you’d like to look up.
