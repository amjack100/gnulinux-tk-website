type

Thu, Sep 23, 2021

type is used for descibing a command as a classification of 5 possible command types:

  • shell built-in
  • function
  • alias
  • hashed command
  • keyword

Otherwise if the command provided is an executable file, then its full path will be displayed.

Usage Examples

Display a command’s type.

Display all the locations of an executable.

$ type -p python

Notes

  • command -V provides similar functionality to type.
