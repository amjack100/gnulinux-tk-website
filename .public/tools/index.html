Tools

Complex

Highly complicated tools usually with either hundreds of configuration options or their own language syntax. One should approach these tools with a mindset on par with learning a new programming language.

make
ffmpeg
cmake

Non-basic

Tools which require some experience to use effectively. These might have configuration files made up of multiple data structures or take advantage of background services.

apt
sed
systemd

Basic

Tools that take a few options and rarely have a manual longer than a couple pages. These include most built-in commands and can be picked up after a few dozen uses.

cp
mkdir / rmdir
update-initramfs
grep
lsmod
cat / tac
ls / dir / vdir
find
ln
chown
uname
chmod
du
tput
touch
man
rsync
ssh
