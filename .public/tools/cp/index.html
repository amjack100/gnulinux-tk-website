cp
Package GNU Coreutils
Usage Filesystem

cp is the standard copy command packaged into Linux systems.

Usage Examples

Copy a single file to another file

cp file.txt copy.txt

Copy files to a directory

cp file.txt directory

Copy an entire directory tree to another location

cp -r source-dir destination-dir
# where source-dir might be a structure like:
#source-dir/
#└── one
#    └── two

Configuration

Backup suffixes defined through environment variables
