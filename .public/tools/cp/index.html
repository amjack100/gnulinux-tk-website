cp

Wed, Aug 11, 2021

cp is the standard copy command packaged into most Linux systems.

Most Common Usages

Copy a single file to a given location

cp file.txt copy.txt

Copy an entire directory tree to another location

cp -r source-dir destination-dir
# where source-dir might be a structure like:
#source-dir/
#└── one
#    └── two
