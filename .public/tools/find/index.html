find
Package GNU Findutils
Usage Searching

find is the primary built-in utility for finding files on a GNU/Linux system.

Usage Examples

The first 2 examples begin searching from the current working directory.

Find files that match a search pattern

find -iname '*.txt'

-iname means “case insensitive”, which can in general be more helpful than -name.

Find files that have been modified within a specific time frame

find -cmin -20

Where the last modified time is 20 minutes

Find every file on the entire system which matches a search pattern

sudo find / -iname 'filename*'

Configuration

Minor configuration options can be specified with environment variables. See the find man page for more specific details.

Notes

  • find is frequently used in junction with xargs, especially within scripts.
