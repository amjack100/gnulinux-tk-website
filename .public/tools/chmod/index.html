chmod is a utility for changing the mode (permissions) of a file or directory. To check the file mode for file.txt, use the command ls -l file.txt. The output is:

-rw-rw-r-- 1 user group 23 Jul 2 08:45 file.txt

where -rw-rw-r-- is the file’s mode.

The first dash is the type of file and is not important. The next 9 dashes are mode bits. A letter (r,w,x) means on and a dash means off.

Most Common Usages

Allow a file to be executed by everyone.

chmod +x file.txt 

# same as chmod a+x file.txt

Give specific permissions to specific people.

chmod u=rwx file.txt

# u (user), g (group), o (other), or a (all)

Assign permissions to a file using octal values

chmod 777 file.txt

# same as chmod +rwx

Notes

Octal value permissions can be translated with the following table:

# Permissions Label
0 None ---
1 Execute --x
2 Write -w-
3 Execute & Write -wx
4 Read r--
5 Read & Execute r-x
6 Read & Write rw-
7 All rwx

Notice how execute (1) + write (2) = 3, execute (1) + read (4) = 5, etc. See octal modes on wikipedia.
